# 📖 Q&A 快速参考指南

## Q1：航班号格式支持情况

### ✅ 答案：完全支持

| 格式类型 | 示例 | 状态 |
|---------|------|------|
| 两字母 + 数字 | CA123、BA456 | ✅ 支持 |
| **两字母 + 4位数字** | **MU5147、CA8888** | **✅ 支持** |
| **数字 + 字母 + 数字** | **9C1234、3U8765** | **✅ 支持** |
| 字母 + 多位数字 | BA12345、FM9001 | ✅ 支持 |
| 纯字母 | BA、CZ、FM | ✅ 支持 |
| 纯数字 | 123、5678 | ✅ 支持 |

### 正则表达式

```
[A-Z0-9]+

含义：一个或多个大写字母或数字
```

### 实际测试用例

✅ **都能正确解析**：

```
乘坐CA123 北京-上海 T1 当地时间10:30-13:45 【航旅纵横】
    ↓ 解析 ↓
[FLIGHT] CA123 PEK-PVG T1 T3｜Local Time 10:30-13:45 [CTZ]

乘坐MU5147 北京-上海 T1 当地时间10:30-13:45 【航旅纵横】
    ↓ 解析 ↓
[FLIGHT] MU5147 PEK-PVG T1 T3｜Local Time 10:30-13:45 [CTZ]

乘坐9C1234 北京-上海 T1 当地时间10:30-13:45 【航旅纵横】
    ↓ 解析 ↓
[FLIGHT] 9C1234 PEK-PVG T1 T3｜Local Time 10:30-13:45 [CTZ]
```

## Q2：读写日历分离

### ✅ 答案：完全支持（两种模式）

### 模式对比

#### 单日历模式（默认）

```
📅 一个日历
├─ 读取：航旅纵横事件
└─ 写入：修改同一事件

特点：简单、快速、原地修改
用途：个人使用、快速转换
```

**使用方法**：
1. 偏好设置 → "单日历模式"
2. 选择包含航旅纵横事件的日历
3. 点击"转换航班事件"

**结果**：事件标题被直接修改

---

#### 分离模式（新增）✨

```
📅 源日历（读取）          📅 目标日历（写入）
├─ 原始事件（保持不变）    └─ 格式化新事件
```

**使用方法**：
1. 偏好设置 → "分离模式"
2. 源日历 = 选择航旅纵横日历
3. 目标日历 = 选择 Flighty 日历（或其他）
4. 可选：启用"复制告警到新事件"
5. 点击"转换航班事件"

**结果**：
- ✅ 源日历原始事件保持不变
- ✅ 目标日历创建新的格式化事件
- ✅ 新事件包含原始信息备注
- ✅ 自动检测和跳过重复事件

### 模式选择建议

| 场景 | 推荐模式 | 原因 |
|------|---------|------|
| 个人快速转换 | 单日历 | 简单、快速 |
| 备份原始事件 | 分离 | 安全隔离 |
| 多日历管理 | 分离 | 灵活组织 |
| 跨日历汇聚 | 分离 | 支持多源 |
| 团队协作 | 分离 | 易于同步 |
| 试验新功能 | 分离 | 易于回滚 |

### 分离模式的优势

```
1️⃣ 安全隔离
   原始事件完全不变
   出错可直接删除目标日历事件回滚

2️⃣ 灵活组织
   从任意日历读取
   写入到任意日历
   支持多源汇聚

3️⃣ 信息完整
   保留原始标题作为备注
   记录来源日历名称
   保留并复制告警

4️⃣ 智能处理
   自动检测重复事件
   避免重复创建
   支持多次运行
```

## 💡 提示和技巧

### 提示 1：模式切换
```
可以在任何时候切换单日历/分离模式
不会影响已有数据
推荐在偏好设置中尝试
```

### 提示 2：重复检测
```
分离模式自动检测重复
标准：相同的格式化标题 = 重复
多次运行不会创建重复事件
```

### 提示 3：告警保留
```
启用"复制告警"：保留原始提醒
禁用"复制告警"：不保留提醒
可根据需要在任何时候切换
```

### 提示 4：备注信息
```
分离模式的新事件包含：
- 📌 原始标题
- 📅 来源日历名称

方便追踪来源和回滚
```

### 提示 5：多次运行
```
单日历模式：重复运行无效（已格式化事件被跳过）
分离模式：重复运行安全（重复检测机制）
```

## ⚠️ 注意事项

### ⚠️ 单日历模式
- 直接修改原始事件标题
- 无法完全恢复（除非手动编辑）
- 建议先备份日历

### ⚠️ 分离模式
- 源日历和目标日历**不能相同**
- 系统会自动验证并提示
- 重复检测基于标题匹配

### ⚠️ 航班号格式
- 支持大写字母和数字
- 小写字母会被忽略（系统级别）
- 特殊字符不支持

### ⚠️ 权限要求
- macOS：需要在系统偏好设置中手动授予日历权限
- 应用首次运行时会提示

## 🔧 故障排除

### 问题：分离模式找不到日历
**原因**：日历已被删除或不可写
**解决**：在偏好设置中重新选择有效的日历

### 问题：事件没有被转换
**原因**：
- 事件标题不符合格式（不含"航旅纵横"）
- 已经被格式化（含"[FLIGHT]"）
- 所选日历中无相关事件

**检查**：查看日志面板了解详细信息

### 问题：告警没有被复制
**原因**：
- "复制告警"选项未启用
- 原始事件没有告警

**检查**：偏好设置中确认选项状态

### 问题：重复检测不工作
**原因**：标题格式不完全相同（空格、大小写等）
**建议**：手动删除重复事件，或调整重复检测逻辑
